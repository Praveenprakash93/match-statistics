# 🏏 IPL Match Statistics Analytics using Google Cloud

## 📌 Project Overview

This project analyzes IPL cricket match data using **Google Cloud Platform (GCP)** to generate meaningful insights into player performance and match outcomes. The solution leverages **BigQuery** for large-scale SQL analytics and **Looker Studio** for interactive dashboards, demonstrating an end-to-end cloud-based data analytics workflow.

## 🚀 Features

* 📊 Identify top batsmen based on **Batting Average** and **Strike Rate**
* 🎯 Analyze the **most economical bowlers** during death overs (16–20)
* 🏆 Compare **winning percentages** when batting first vs. chasing
* 📈 Interactive dashboards built with **Looker Studio**
* ⚡ Fast SQL analytics using **Google BigQuery**

## 🛠️ Tech Stack

* **Google Cloud Platform (GCP)**
* **BigQuery**
* **Google Cloud Storage (Optional)**
* **Looker Studio**
* **SQL**
* **CSV Datasets (Kaggle IPL Dataset / Cricsheet)**

* IPL Matches Dataset (`matches.csv`)

Source:

## 📊 Key Analyses

* Top-performing batsmen by batting average and strike rate
* Most economical bowlers in death overs
* Match-winning trends based on batting order
* SQL aggregations and performance metrics

## 🔄 Project Workflow

1. Import IPL datasets into Google BigQuery.
2. Clean and preprocess the data.
3. Perform SQL-based analytics.
4. Create aggregated views for faster querying.
5. Build interactive dashboards using Looker Studio.
6. Generate insights for players, teams, and match strategies.
